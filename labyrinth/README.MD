# Labyrinth

## Control Keys

```
WASD - movement
Shift - run
Tab - triangle-mode
Esc - exit

Up/Down - change volume
Left/Right - change track
```

## Special folders

```
./res/audio/music - put your music(in *.ogg format) here
Do not use nonLatin alphabets!
```
